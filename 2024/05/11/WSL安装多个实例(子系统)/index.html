  1. 在 Windows 上启用所需的可选组件（Windows Virtualisation Platform 等）
  2. 启用 Windows Subsystem for Linux 2（WSL2）
  3. 将 Linux 内核更新到最新版本
  4. 安装默认的 Linux 发行版，比如最新版的 Ubuntu

安装完成，只需要运行 wsl 命令即可打开 WSL2 shell，第一次运行时需要设置用户名和密码。
